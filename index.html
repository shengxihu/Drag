Drag

An image uploader powered by Nodejs, Electron，Vuejs.

View on GitHub Download .zip Download .tar.gz

什么是Drag?

这是一个支持拖拽上传图片到图床的桌面客户端，国内使用七牛，国外使用cloudinary。你可以使用拖拽或点击上传的方式将本地图片上传到在线图床并点击即可复制链接。

如何使用

获取账户信息

1.国内图床

  • 注册七牛

  • 获取Access/Secret Key

1.国外图床

  • 注册：Cloudinary

  • 获取：Cloud name/API Key/API Secret

配置

备注：

  • 七牛资源域名即存储资源的外链默认域名

  • BucketName即存储资源列表的名称

  • 使用localhost发送请求，使用时请关闭翻墙服务

管理

管理界面暂时只支持国内图床完整文件名搜索🔍.

开发相关

如何开发

  • 克隆项目Drag

  • 安装依赖：npm install

  • 开发：npm run dev

  • 启动：npm run start

  • 打包应用：npm run package

技术架构

  • 应用环境：Electron
  • 前端：Vuejs,Element
  • 后端：Nodejs
  • 数据本地化：nconf读取本地json配置文件，localstorage数据持久化
